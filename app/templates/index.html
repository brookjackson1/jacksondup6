Welcome to The Daily News

A curated collection of the latest headlines and trending stories from around the world.

About This Project

This Flask web application integrates with News API to deliver up-to-date news and trending stories from thousands of sources worldwide. The platform allows you to browse top headlines, search articles by keyword or category, and manage your own collection of saved articles.

Built as part of Module 6 coursework, this application demonstrates full-stack development skills including API integration, database management, and responsive web design.

Features

  • → Top Headlines: View the latest breaking news and top stories from trusted sources
  • → Search & Filter: Find specific news by keyword or browse by category (business, sports, technology, etc.)
  • → Save Favorites: Create your personal collection with full CRUD operations (Create, Read, Update, Delete)
